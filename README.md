Note: I'm not using the browser at all, everything runs through `elm-test`


## Run the tests!

```
# Install npm dependencies
$ npm install

# Install elm, elm-test-rs, elm-format, elm-json locally too (through elm-tooling)
$ npx elm-tooling install

# Run the test suite
$ npm test
```
